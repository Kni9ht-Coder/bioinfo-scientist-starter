# AGENTS.md

## Project identity

This repository is a Codex-assisted bioinformatics / AIDD manuscript system.
The goal is reproducible science, not fluent unsupported writing.

## Human authority

The human researcher and supervisor retain final authority over:
- research questions
- biological interpretation
- ethics and privacy
- authorship
- journal submission
- response to reviewers

Codex must not submit papers, upload private data, contact journals, or make final scientific decisions.

## Non-negotiable rules

1. Never fabricate citations, DOI/PMID/accession numbers, datasets, p-values, model metrics, biological mechanisms, wet-lab validation, or clinical conclusions.
2. Do not write manuscript claims unless supported by `results/claim_ledger.yml`, real metrics, generated tables/figures, or verified references.
3. If evidence is missing, write `TODO: evidence needed`.
4. Every experiment must record command, config, input data version/hash, software environment, seed, git commit hash, and output location.
5. Human/clinical data must not be committed or exposed to external services.
6. For ML/AIDD tasks, check leakage and use fair baselines.
7. For omics statistics, check batch/confounding and use multiple-testing correction where applicable.
8. Figures must be generated by code from data/results.

## Standard commands

- Install: `pip install -e .`
- Lint: `make lint`
- Format: `make format`
- Test: `make test`
- Smoke: `make smoke`
- Snakemake dry run: `make snakemake-dry`
- Figures: `make figures`
- Audit: `make audit`
- Paper HTML: `make paper-html`

## Expected Codex response format

For every coding or writing task, report:
1. Files changed
2. Commands run
3. Tests/checks passed or failed
4. Scientific assumptions made
5. Evidence used
6. Unverified items
7. Suggested next issue

## Task discipline

Prefer small pull requests: one data loader, one QC step, one baseline, one figure, one manuscript subsection, or one audit script.
