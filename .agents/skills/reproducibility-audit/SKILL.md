Checklist:
1. Environments can be created.
2. Raw data sources documented.
3. Processed data generated by pipeline.
4. Figures generated by scripts.
5. Claims in claim_ledger.
6. Citations verified.
7. Seeds and package versions recorded.
8. CI passes.
9. AI usage disclosed.
10. Private data excluded.

Output docs/reproducibility_audit.md and docs/submission_checklist.md.
