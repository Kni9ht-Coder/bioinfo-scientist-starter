# Submission checklist

- [ ] Human approval of research question
- [ ] Human approval of data sources and inclusion criteria
- [ ] Human approval of experiment protocol
- [ ] All verified claims have evidence paths
- [ ] Citations are verified against paper cards and BibTeX
- [ ] Figures are generated by code from data or results
- [ ] Reproducibility audit passed
- [ ] AI use is disclosed where required
- [ ] Private or identifiable human data is excluded from Git
